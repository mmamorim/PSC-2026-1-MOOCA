# Exercícios - Aula 05 - 24/03

#### Exercício 1 – Contagem de Caracteres
>
> Crie um programa que:
> 
> 1. Solicite uma palavra ou frase ao usuário.  
> 2. Exiba a quantidade de caracteres digitados (incluindo espaços).
> 3. Exiba o primeiro e o último caractere  
> 4. Exiba o texto em letras maiúsculas e minúsculas

#### Exercício 2 – Contador de Vogais

> Crie um programa que:
> 
> 1. Solicite uma frase ao usuário.  
> 2. Percorra a String caractere por caractere.  
> 3. Conte quantas vogais existem (`a, e, i, o, u`).  
> 4. Exiba o total de vogais.
> 
> 📌 **Desafio:**  
> Considere letras maiúsculas e minúsculas.

#### Exercício 3 – Inversão de Texto
> Crie um programa que:
> 
> 1. Solicite uma palavra ao usuário.  
> 2. Utilize um laço para percorrer a palavra de trás para frente.  
> 3. Exiba a palavra invertida.
> 
> **Exemplo**
> 
> Entrada: `Java`  
> Saída: `avaJ`

#### Exercício 4 - Verificação de Palíndromo
> Crie um programa que:
> 
> 1. Solicite uma palavra.  
> 2. Verifique, utilizando um laço, se ela é um **palíndromo**  
   (ou seja, se é igual quando lida de trás para frente).
> 
> **Exemplos**
> 
> - `arara` → palíndromo  
> - `java` → não é palíndromo  

#### Exercício 5
> Ler 3 números inteiros e apresentá-los em ordem crescente.


#### Exercício 6 – Validação de Senha
> 
> Crie um programa que:
> 
> 1. Solicite uma senha ao usuário.  
> 2. Verifique se a senha atende às seguintes regras:
> 
> - possui pelo menos 8 caracteres  
> - contém pelo menos um número  
> - contém pelo menos uma letra maiúscula  
> 
> 3. Exiba se a senha é válida ou inválida.
> 
> 📌 **Desafio:**  
> Mostre qual regra não foi atendida.
